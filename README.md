# Saprikina FitSystem

Лендинг персонального тренера — **Мария Сапрыкина**, сертифицированный эксперт FPA.  
Сайт презентует услуги фитнес-тренинга, результаты клиентов и отзывы.

---

## Стек

| Технология | Роль |
|---|---|
| HTML5 | единственный файл — `index.html` |
| [Tailwind CSS](https://tailwindcss.com/) (CDN) | утилитарная стилизация |
| Vanilla JS | анимации, мобильное меню, модалка отзывов |
| Google Fonts — Inter | типографика |

Сборка не требуется. Сайт работает как статическая страница.

---

## Структура страницы

### Навигация
Фиксированный хедер с glassmorphism-эффектом.  
Ссылки: **Услуги → Результаты → Отзывы → Связаться**.  
На мобильных — бургер-меню с анимацией.

### Hero
Главный экран с заголовком **«Фитнес как система: результат на основе данных»**.  
Справа — data-виджеты:
- кольцо прогресса (90% клиентов достигают цели)
- бейдж результата (100% Success)
- верифицированный сертификат FPA
- анимированный бар-чарт динамики за 12 недель

### Marquee (бегущая строка)
Перечисляет преимущества: сертификат FPA, расчёт КБЖУ, гайд по питанию, разбор техники, online 24/7, тренировки в зале и дома.

### Услуги (`#services`)
Три карточки форматов работы:

| # | Формат | Особенность |
|---|---|---|
| 01 | Офлайн — персональное ведение в зале | 1-я тренировка бесплатно |
| 02 | Онлайн 1-на-1 по видеосвязи | Популярный · чат 24/7 |
| 03 | Индивидуальный план для самостоятельных | Видео-анализ техники |

Все форматы включают расчёт КБЖУ и гайд по питанию.

### Разбор техники (`#technique`)
Видео-плейсхолдер с Demo-плашкой.  
Описание процесса: анализ видео за 24 ч, подробные комментарии, корректировка программы еженедельно.

### Результаты (`#results`)
Блок «До / После» с метриками примера: **−5 кг** веса, **−7 см** талии за **1 месяц**.

### Отзывы (`#testimonials`)
Четыре карточки отзывов (Алина, Дарья, Анастасия ×2), каждый — 5 звёзд.  
Кнопка «Посмотреть оригинал» открывает **модальное окно** с фото скриншота отзыва.

### Квалификация (`#expertise`)
Сертификат FPA · нутрициология · 3+ года практики · 100+ клиентов.

### CTA / Контакт (`#contact`)
Первая тренировка в зале — бесплатно.  
Кнопки: **Telegram** и **Instagram**.

### Футер
Копирайт 2026 · ссылки на соцсети.

## Деплой на GitHub Pages

1. Убедиться, что SSH-конфиг настроен (`~/.ssh/da_roadmap_key`).
2. Инициализировать репозиторий и запушить:
   ```bash
   git init
   git add .
   git commit -m "init"
   git remote add origin git@github.com:konevs/fitness-site.git
   git push -u origin main
   ```
3. В настройках репозитория: **Settings → Pages → Branch: main / (root) → Save**.
4. Сайт будет доступен по адресу: `https://konevs.github.io/fitness-site/`
